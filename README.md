# teachme
基于HoshinoBotV2的maimei和Arcaea教教表情包生成器
## 使用方法
1. 将该项目放在Hoshinobot插件目录`modules`下，或者clone本项目
2. 修改`teachme.py`中的资源文件目录
3. 在`config/__bot__.py`模块列表中添加 `teachme`
4. 重启Hoshinobot

## 指令
| 命令             | 功能       |
|------------------|-----------|
| 教教id<歌曲编号>  | maimai歌曲 |
| 教教<歌曲名>      | arcaea歌曲 |
